for getting the x-acess-token(JWT):
curl -X GET https://secret-headland-55348.herokuapp.com/login -H 'authorization: Basic YWRtaW46MTIzNDU=' -d '{"name":"admin","password":"12345"}'

for bank details with IFSC :
curl -X GET https://secret-headland-55348.herokuapp.com/bank/ABHY0065002 -H 'x-access-token: *insert token here*'

for bank details with name w/o limit and offset params:
curl -X GET https://secret-headland-55348.herokuapp.com/bank/ALLAHABAD%20BANK/ALIGARH -H 'x-access-token: *insert token here*'

for bank details with name with limit and offset params:(first one is limit and second offset)
curl -X GET https://secret-headland-55348.herokuapp.com/bank/ALLAHABAD%20BANK/ALIGARH/5/2 -H 'x-access-token: *insert token here*'
